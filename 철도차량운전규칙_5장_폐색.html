🚄 철도 폐색방식 완전 가이드

철도차량운전규칙 제5장 - 열차간의 안전확보

📚 폐색방식이란?

폐색(閉塞)은 열차의 충돌이나 추돌을 방지하기 위해 하나의 구간에 한 대의 열차만 운행할 수 있도록 선로를 구간별로 나누어 관리하는 안전 시스템입니다. 마치 도로의 신호등처럼 철도에서 교통을 통제하는 핵심 기술입니다.

🔧 상용폐색방식 (평상시 사용)

1자동폐색식 (ABS - Automatic Block System)

가장 현대적이고 안전한 방식으로, 궤도회로를 통해 열차의 위치를 자동으로 감지하여 신호기를 제어합니다.

A역
B역
🚊
폐색구간 1
폐색구간 2

🤖 완전 자동화

사람의 개입 없이 자동으로 열차 위치 감지 및 신호 제어

⚡ 실시간 감지

궤도회로로 열차 위치를 즉시 파악하여 안전 확보

🎯 높은 효율성

운행 횟수 증대 및 합리적 열차 운용 가능

2연동폐색식 (Controlled Manual Block System)

역간을 하나의 폐색구간으로 하여 신호기와 폐색장치를 연동시켜 운영하는 방식입니다.

출발역
도착역
👨‍💼 역무원이 폐색 승인 버튼 조작

🔗 이중 안전장치

신호기와 폐색장치가 연동되어 둘 다 정상이어야 출발 가능

👥 상호 협력

출발역과 도착역 역무원의 상호 승인으로 안전 확보

3차내신호폐색식 (Cab Signal Block System)

ATC 구간에서 사용되며, 신호를 기관사의 운전실 내부로 전송하여 표시하는 방식입니다.

🚊
속도 80km/h
차내 신호 디스플레이

📱 차내 표시

운전실 ADU를 통해 허용 속도 및 신호 현시

🛡️ 자동 제동

허용 속도 초과 시 자동으로 제동장치 작동

🔄 실시간 업데이트

앞 열차와의 간격에 따라 지시 속도 자동 조절

4통표폐색식 (Tablet Block System)

단선 구간에서 사용되는 기계적 안전장치로, 물리적인 통표(토큰)를 이용한 방식입니다.

A역
B역

🎫

🎫 물리적 토큰

통표를 가진 열차만 해당 구간 운행 가능

🔒 기계적 안전

통표가 삽입되지 않으면 다음 폐색 수속 불가

🚇 단선 전용

단선 기계연동장치 구간에서만 사용

⚠️ 대용폐색방식 (비상시 사용)

대용폐색방식은 상용폐색방식을 사용할 수 없는 비상상황(장비 고장, 사고, 보수작업 등)에서 임시로 적용하는 방식입니다.

📡지령식

CTC 구간에서 관제사가 직접 승인하여 열차를 운행시키는 방식

🎮 관제사 직접 제어

관제실에서 모니터로 열차 위치 확인 후 운행 승인

📞통신식

전화나 무선통신을 통해 역 간 연락하여 폐색을 확인하는 방식

☎️ 음성 확인

역무원 간 직접 통화로 안전 확인

📋지도통신식

지도표라는 임시 통행권을 발행하여 운행하는 방식 (통표폐색의 대용)

📄 임시 통행권

지도표 또는 지도권으로 통표 역할 대체
